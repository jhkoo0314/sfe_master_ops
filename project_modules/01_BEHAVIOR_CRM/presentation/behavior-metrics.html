HIR (High-Impact Rate) - 정직입력률

행동 기록의 품질을 측정하는 지표

📝 정의

Behavior Score의 quality_score를 기반으로 계산되는 지표로, 사용자가 입력한 활동 기록의 정확성과 품질을 평가합니다.

🎯 목적

데이터의 신뢰성을 보장하고, 정확한 행동 분석을 위한 기초 데이터 품질을 측정합니다.

계산 공식

HIR = 평균(각 Behavior 타입별 quality_score)

단계별 계산:
1. 최근 30일간의 Behavior Score 조회
2. 8개 Behavior 타입별로 가장 최근 quality_score 추출
3. 추출된 quality_score들의 평균 계산
4. 0-100 스케일로 정규화

예시 계산

최근 30일간의 Behavior Score에서:

  • Call: quality_score = 85
  • Visit: quality_score = 90
  • Email: quality_score = 80
  • Presentation: quality_score = 95
  • ... (8개 타입)

HIR = (85 + 90 + 80 + 95 + ...) / 8 = 87.5점

대시보드 표시

  • 위치: 메인 대시보드 - ProfileCard (왼쪽 상단)
  • 표시 형식: "HIR (정직) 87%" 형태로 표시
  • 색상: 녹색 계열 (높을수록 좋음)
  • 변화 추적: 이전 기간 대비 증감 표시 (▲/▼)

해석 가이드

점수 범위 의미 권장 조치
90-100 매우 높은 품질 현재 수준 유지
70-89 양호한 품질 일부 개선 여지 있음
50-69 보통 품질 기록 정확도 개선 필요
0-49 낮은 품질 즉시 개선 필요
🌡️

RTR (Relationship Temperature Rate) - 관계온도

고객과의 관계 품질을 측정하는 지표

📝 정의

Activity의 sentiment_score를 기반으로 계산되는 지표로, 고객과의 관계가 얼마나 긍정적인지를 측정합니다.

🎯 목적

고객 관계의 건강도를 파악하고, 관계 개선이 필요한 고객을 식별합니다.

계산 공식

RTR = 평균(Activity의 sentiment_score)

단계별 계산:
1. 최근 30일간의 Activity 조회
2. sentiment_score가 있는 Activity만 필터링
3. sentiment_score의 평균 계산
4. 0-100 스케일로 정규화

예시 계산

최근 30일간의 Activity:

  • Activity 1: sentiment_score = 85
  • Activity 2: sentiment_score = 90
  • Activity 3: sentiment_score = 75
  • Activity 4: sentiment_score = null (제외)
  • Activity 5: sentiment_score = 95

RTR = (85 + 90 + 75 + 95) / 4 = 86.25점

대시보드 표시

  • 위치: 메인 대시보드 - ProfileCard
  • 표시 형식: "RTR (관계) 86점" 형태로 표시
  • 색상: 노란색 계열 (관계 온도 표현)
  • 시각화: 레이더 차트에서 오른쪽 축

해석 가이드

점수 범위 관계 상태 권장 조치
80-100 매우 긍정적 관계 유지 및 강화
60-79 양호한 관계 지속적인 관리
40-59 보통 관계 관계 개선 활동 필요
0-39 관계 악화 즉시 관계 회복 조치
🔄

BCR (Behavior Consistency Rate) - 루틴 점수

행동의 일관성과 규칙성을 측정하는 지표

📝 정의

활동 빈도, 규칙성, Behavior Score 일관성을 종합하여 계산되는 지표로, 꾸준한 활동 패턴을 평가합니다.

🎯 목적

일관된 업무 패턴을 유지하는지 확인하고, 루틴이 깨진 경우를 조기에 발견합니다.

계산 공식

BCR = 빈도 점수(40점) + 규칙성 점수(30점) + 품질 일관성 점수(30점)

세부 계산:
1. 빈도 점수 = (활동 일수 / 총 일수) × 40
2. 규칙성 점수 = 활동 간격의 표준편차 기반 점수 × 30
3. 품질 일관성 = Behavior Score quality_score 표준편차 기반 점수 × 30
4. 총점 = 빈도 + 규칙성 + 품질 일관성 (최대 100점)

예시 계산

최근 30일간:

  • 활동 일수: 20일 / 총 30일 → 빈도 점수 = (20/30) × 40 = 26.7점
  • 활동 간격 표준편차 낮음 → 규칙성 점수 = 25점
  • quality_score 표준편차 낮음 → 품질 일관성 = 28점

BCR = 26.7 + 25 + 28 = 79.7점

대시보드 표시

  • 위치: 메인 대시보드 - ProfileCard
  • 표시 형식: "BCR (루틴) 80%" 형태로 표시
  • 색상: 기본 색상
  • 시각화: 레이더 차트에서 하단 축

해석 가이드

점수 범위 일관성 수준 권장 조치
80-100 매우 일관적 현재 패턴 유지
60-79 양호한 일관성 일부 개선 여지
40-59 보통 일관성 규칙적인 패턴 형성 필요
0-39 일관성 부족 루틴 개선 필수
📅

PHR (Proactive Health Rate) - 선제적 관리 점수

다음 활동 계획의 적절성을 측정하는 지표

📝 정의

Activity의 next_action_date를 기반으로 계산되는 지표로, 고객을 얼마나 선제적으로 관리하고 있는지 평가합니다.

🎯 목적

Dead Lead를 방지하고, 적절한 시기에 고객을 관리하여 관계를 유지합니다.

계산 공식

PHR = 평균(각 Activity의 관리 점수)

Activity별 점수 기준:
- next_action_date가 지남 (Dead Lead): 0점
- 0-7일 이내: 100점 (적극적 관리)
- 8-14일: 80점
- 15-30일: 60점
- 30일 이상: 40점
- next_action_date 없음: 0점

예시 계산

최근 30일간의 Activity:

  • Activity 1: next_action_date = 3일 후 → 100점
  • Activity 2: next_action_date = 10일 후 → 80점
  • Activity 3: next_action_date = 20일 후 → 60점
  • Activity 4: next_action_date 없음 → 0점
  • Activity 5: next_action_date = 지난 날짜 → 0점

PHR = (100 + 80 + 60 + 0 + 0) / 5 = 48점

대시보드 표시

  • 위치: 메인 대시보드 - ProfileCard
  • 표시 형식: "PHR (관리) 48%" 형태로 표시
  • 색상: 기본 색상
  • 시각화: 레이더 차트에서 왼쪽 축

해석 가이드

점수 범위 관리 수준 권장 조치
80-100 매우 적극적 현재 관리 수준 유지
60-79 양호한 관리 일부 개선 여지
40-59 보통 관리 다음 활동 계획 수립 필요
0-39 관리 부족 즉시 Dead Lead 점검 및 관리
⭐

Total Score - 종합 점수

4개 지표의 평균 종합 점수

계산 공식

Total Score = (HIR + RTR + BCR + PHR) / 4

예시:
HIR = 87, RTR = 86, BCR = 80, PHR = 48
Total Score = (87 + 86 + 80 + 48) / 4 = 75.25점

랭크 시스템

Total Score 랭크 상위 비율
90-100 🏆 Excellent 상위 5%
80-89 🏆 Excellent 상위 12%
70-79 ⭐ Good 상위 30%
60-69 📊 Average 상위 50%
0-59 📈 개선 필요 하위 50%
