Team Behavior Score

팀 전체 평균 Behavior Score

📝 정의

팀원들의 개별 Behavior Score를 평균하여 계산한 팀 전체 점수입니다.

🎯 목적

팀 전체의 행동 품질 수준을 파악하고, 개선이 필요한 팀원을 식별합니다.

계산 공식

Team Behavior Score = 평균(각 팀원의 Behavior Score)

계산 단계:
1. 팀원 목록 조회 (getTeamMembers)
2. 각 팀원의 Behavior Score 계산 (최근 30일)
- 8개 Behavior 타입별 quality_score 평균
3. 팀원별 점수 합산 후 평균 계산
4. 이전 기간 대비 변화율 계산

예시 계산

팀원 5명의 Behavior Score:

  • 팀원 A: 85점
  • 팀원 B: 78점
  • 팀원 C: 92점
  • 팀원 D: 80점
  • 팀원 E: 88점

Team Behavior Score = (85 + 78 + 92 + 80 + 88) / 5 = 84.6점

이전 기간: 82점

변화율 = ((84.6 - 82) / 82) × 100% = +3.2%

대시보드 표시

Team Behavior Score
85점
▲ 전월 대비 3점 상승
🎯

Avg. HIR (정직입력)

팀원들의 평균 HIR 점수

계산 공식

Avg. HIR = 평균(각 팀원의 HIR 점수)

각 팀원의 HIR 계산:
- 최근 30일간의 Behavior Score 조회
- 8개 Behavior 타입별 quality_score 평균
- 0-100 스케일로 정규화

예시 계산

팀원 5명의 HIR:

  • 팀원 A: 87%
  • 팀원 B: 75%
  • 팀원 C: 95%
  • 팀원 D: 82%
  • 팀원 E: 90%

Avg. HIR = (87 + 75 + 95 + 82 + 90) / 5 = 85.8%

해석 가이드

Avg. HIR 범위 의미 권장 조치
90-100% 신뢰도 매우 높음 현재 수준 유지
70-89% 신뢰도 높음 일부 개선 여지
0-69% 신뢰도 개선 필요 데이터 입력 품질 개선 교육

팀 전체 파이프라인 금액

계산 공식

Team Goal Forecast

팀 목표 달성 예측률

계산 공식

Team Goal Forecast = (Avg. HIR / 목표 HIR) × 100%

현재 로직:
- 목표 HIR = 70점 (기본값)
- Avg. HIR을 목표 대비 비율로 계산
- 최대 100%로 제한

예시 계산

Avg. HIR = 85.8%

목표 HIR = 70%

Team Goal Forecast = (85.8 / 70) × 100% = 122.6% → 100% (최대값 제한)

해석 가이드

Forecast 범위 의미 상태
100% 이상 목표 달성 ✅ 목표 달성
90-99% 목표 달성 임박 ⚠️ 목표 달성 필요
0-89% 목표 달성 필요 ❌ 개선 필요

Coaching Priority List

코칭 우선순위가 높은 팀원 목록

우선순위 결정 기준

우선순위 점수 계산

  • Behavior Score가 낮은 팀원
  • HIR이 낮은 팀원 (데이터 신뢰도 낮음)
  • 최근 성과가 하락한 팀원
  • 4개 지표 중 균형이 깨진 팀원

대시보드 표시

CoachingPriorityList 컴포넌트

팀원별로 다음 정보 표시:

  • 이름 및 프로필
  • Behavior Score
  • 주요 개선 영역 (가장 낮은 지표)
  • 코칭 액션 제안

Team Balance Radar

팀 전체의 지표 밸런스 분석

계산 방법

팀 평균 지표 = 평균(각 팀원의 개별 지표)

계산 지표:
- 팀 평균 HIR
- 팀 평균 RTR
- 팀 평균 BCR
- 팀 평균 PHR

레이더 차트로 시각화하여
팀 전체의 강점/약점 영역 파악

해석 방법

  • ✓ 레이더 차트가 균형잡힌 형태 → 팀 전체가 균형있게 발전
  • ✓ 특정 축이 낮음 → 해당 영역 집중 코칭 필요
  • ✓ 모든 축이 높음 → 팀 전체 성과 우수
