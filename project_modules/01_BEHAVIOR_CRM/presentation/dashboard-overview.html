메인 대시보드 (Action Hub)

행동 데이터 기반 대시보드

레이아웃 구조

ProfileCard
(2행)
BehaviorRadarChart
(2행)
ActionQueue
(2행)
BehaviorOutcomeScatter
(1행)

주요 컴포넌트

ProfileCard

  • 사용자 프로필 (아바타, 이름)
  • Total Score (큰 숫자)
  • 랭크 배지 (상위 5%, 12% 등)
  • 4개 지표 그리드 (HIR, RTR, BCR, PHR)
  • 이전 기간 대비 변화 표시

BehaviorRadarChart

  • SVG 기반 레이더 차트
  • 4개 지표 시각화 (HIR, RTR, BCR, PHR)
  • 가장 낮은 지표 팁 메시지
  • 균형 분석 및 개선 제안

ActionQueue

  • 우선순위별 액션 목록
  • Dead Lead 알림
  • 다음 활동 예정일 관리
  • 즉시 조치 필요한 항목 강조

BehaviorOutcomeScatter

  • 행동 점수 vs 성과 스캐터 차트
  • 인과관계 시각화
  • 트렌드 라인
  • 데이터 포인트 상호작용
💰

성과 리포트 (Outcome Report)

매출 통계 및 성과 분석

레이아웃 구조

상단: KPI 카드 4개 (Revenue, Growth, Pipeline, Prediction)

하단: 2열 그리드

RevenueBehaviorChart
(메인 차트)
GrowthLeaderboard
PipelineFunnel

주요 컴포넌트

RevenueKpiCard

  • 총 매출액 표시
  • 목표 대비 달성률
  • 진행 바 (Progress Bar)
  • 목표까지 남은 금액

GrowthKpiCard

  • YoY 성장률 표시
  • 업계 평균 대비 비교
  • 색상 코딩 (양수/음수)

RevenueBehaviorChart

  • 매출 트렌드 라인 차트
  • 행동 점수 오버레이
  • 인과관계 시각화
  • 시간대별 필터링

GrowthLeaderboard

  • 상위 성장 고객 목록
  • 성장률 순위
  • 고객별 상세 정보

PipelineFunnel

  • 파이프라인 단계별 집계
  • 퍼널 시각화
  • 전환율 표시
  • 병목 지점 식별
👥

관리자 대시보드 (Coaching Cockpit)

팀 관리 및 코칭 대시보드

레이아웃 구조

상단: CoachingCockpitHeader

중간: TeamKpiRow (4개 KPI 카드)

하단: 2열 그리드

CoachingPriorityList
(코칭 우선순위)
TeamBalanceRadar
MarketIntel

주요 컴포넌트

TeamKpiRow

  • Team Behavior Score
  • Avg. HIR (정직입력)
  • Active Pipeline
  • Team Goal Forecast
  • 각 KPI별 트렌드 표시

CoachingPriorityList

  • 코칭 우선순위가 높은 팀원 목록
  • 팀원별 Behavior Score
  • 주요 개선 영역 표시
  • 코칭 액션 제안

TeamBalanceRadar

  • 팀 전체 지표 밸런스
  • 4개 지표 평균 시각화
  • 강점/약점 영역 파악

MarketIntel

  • 시장 정보 및 인사이트
  • 업계 평균 대비 비교
  • 시장 트렌드
🌱

성장 맵 (Growth Map)

행동 품질과 성과의 인과관계 분석

주요 컴포넌트

GamificationTier

  • 레벨/경험치 시스템
  • 현재 티어 표시
  • 다음 티어까지 진행률
  • 게이미피케이션 요소

BehaviorOutcomeTrendChart

  • Dual-Axis 트렌드 차트
  • 행동 점수 vs 성과
  • 인과관계 시각화
  • 시간대별 트렌드

SkillTree

  • 3단계 성장 로드맵
  • 달성한 스킬 표시
  • 다음 단계 제안
  • 성장 경로 시각화
🎨

공통 UI 패턴

일관된 디자인 시스템

컴포넌트 스타일

  • 카드 기반 레이아웃: 모든 지표는 Card 컴포넌트로 표시
  • 로딩 상태: Skeleton 컴포넌트로 로딩 중 표시
  • 에러 처리: 에러 발생 시 사용자 친화적 메시지
  • 색상 코딩: 지표별 의미에 맞는 색상 사용
  • 반응형 디자인: 모바일/태블릿/데스크톱 대응

색상 시스템

지표 색상 의미
HIR 녹색 정직입력, 신뢰도
RTR 노란색 관계 온도
BCR 파란색 루틴, 일관성
PHR 보라색 선제적 관리
Revenue 녹색 매출, 성장
