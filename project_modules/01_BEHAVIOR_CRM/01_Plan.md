# Behavior-Driven CRM v2.1 - 사업 기획안

## 📋 Executive Summary

**Behavior-Driven CRM v2.1**은 영업사원의 실제 행동 데이터를 기반으로 성과를 분석하고, 데이터 기반 코칭을 제공하는 혁신적인 CRM 플랫폼입니다. 기존 CRM이 "얼마나 일했는지"만 측정하는 것과 달리, "어떻게 일했기 때문에 성과가 나왔는지"를 구체적으로 분석하여 영업 생산성을 혁신합니다.

---

## 🎯 Problem Statement

### 기존 CRM의 근본적 한계

**1. 입력 중심, 분석 부재**
- 영업사원이 매일 보고서를 작성하느라 현장 업무 시간이 감소
- 기록된 데이터는 단순 방문 횟수나 메모 수준에 머무름
- 관리자는 데이터를 활용하기보다는 '감'에 의존한 의사결정을 내림

**2. 행동의 질적 측정이 불가능**
- "많이 뛰었다"는 양적 평가만 가능
- "어떻게 뛰었는지"에 대한 구체적인 분석 부재
- 효과적인 행동 패턴을 표준화할 수 없음

**3. 성과의 원인 분석 불가**
- 결과는 나오지만 "왜 그 결과가 나왔는지" 설명 불가
- 우수 사원의 성공 요인을 체계적으로 파악할 수 없음
- 개선 방향을 구체적으로 제시할 수 없음

## 🏗️ Solution Architecture

### 아키텍처 개요

Behavior-Driven CRM은 **3계층 아키텍처**로 구성됩니다:

```
┌─────────────────┐
│   Behavior      │ ← 영업사원의 실제 행동 측정
│   Layer         │
├─────────────────┤
│   Analytics     │ ← 행동-성과 상관관계 분석
│   Engine        │
├─────────────────┤
│   Outcome       │ ← 행동의 결과로서의 성과 측정
│   Layer         │
└─────────────────┘
```

### 1. Behavior Layer: 행동 측정의 핵심

#### 8가지 핵심 행동 지표 (Behavior Types)

Behavior-Driven CRM은 영업 프로세스를 **8가지 표준화된 행동 유형**으로 분류합니다:

| 행동 유형 | 영문 | 설명 | 측정 포인트 |
|----------|------|------|-------------|
| **접근** | Approach | 고객과의 첫 만남, 초기 접촉 | 첫 상호작용의 자연스러움 |
| **컨택** | Contact | 지속적인 소통, 관계 구축 | 소통 빈도와 일관성 |
| **대면** | Visit | 직접 방문 및 상담 | 방문의 목적성 및 깊이 |
| **프레젠테이션** | Presentation | 제품/서비스 소개 | 메시지 전달의 명확성 |
| **질문/탐색** | Question | 고객 니즈 파악을 위한 질문 | 질문의 전략성과 깊이 |
| **필요성 자극** | Need Creation | 고객의 잠재적 니즈 발굴 | 니즈 창출의 창의성 |
| **시연/자료제공** | Demonstration | 제품 시연 및 자료 제공 | 시연의 효과성과 타겟팅 |
| **후속관리** | Follow-up | 방문 후 관리 및 관계 유지 | 후속 조치의 적시성 |

#### 행동 측정 방식

각 행동은 **양(Quantity) + 질(Quality)**의 이중 측정 체계로 평가됩니다:

**양적 측정 (Quantity)**
- 빈도: 일정 기간 내 해당 행동의 발생 횟수
- 강도: 행동의 지속 시간 및 집중도
- 일관성: 규칙적인 실행 정도

**질적 측정 (Quality)**
- 전략성: 행동의 목적성과 계획성
- 효과성: 행동의 실제 성과 기여도
- 적응성: 상황별 최적화 정도

### 2. Analytics Engine: 지능적 분석 코어

#### 상관관계 분석 알고리즘

행동 데이터와 성과 데이터를 연결하는 **Pearson 상관계수 기반 분석**을 수행합니다:

```
Correlation = Σ((Xᵢ - X̄)(Yᵢ - Ȳ)) / √[Σ(Xᵢ - X̄)² Σ(Yᵢ - Ȳ)²]
```

- **X**: 특정 행동의 품질 점수
- **Y**: 성과 지표 (HIR, 전환률 등)
- **결과**: -1 ~ +1 사이의 상관계수 (1에 가까울수록 강한 양의 상관관계)

#### Behavior Score 계산 로직

각 Behavior Type별로 **3차원 점수**를 계산합니다:

**1. 행동 강도 점수 (Intensity Score)**
```
Intensity Score = (가중치 합계 / 최대 기대치) × 100

가중치:
- 방문(Visit): 3점
- 전화(Call): 2점
- 메시지(Message): 1점
- 프레젠테이션(Presentation): 2점
- 후속관리(Follow-up): 1점
```

**2. 행동 다양성 점수 (Diversity Score)**
```
Diversity Score = (실행한 행동 종류 수 / 전체 행동 종류 수) × 100
```

**3. 행동 질 점수 (Quality Score)**
```
Quality Score = (평균 품질 점수 × 0.4) + (평균 양 점수 × 0.3) + (후속 비율 × 30)
```

### 3. Outcome Layer: 성과 측정의 종합

#### 4가지 핵심 성과 지표 (KPI)

| 지표 | 영문 | 계산 방식 | 의미 |
|------|------|----------|------|
| **고효과 행동 비율** | HIR (High-Impact Rate) | 고품질 행동의 비율 × 평균 영향력 | 효과적인 행동의 실행 정도 |
| **전환률** | Conversion Rate | (행동 → 매출 전환 성공 건수 / 총 행동 건수) × 100 | 행동의 실제 성과 기여도 |
| **필드 성장률** | Field Growth Rate | ((현재 기간 매출 - 이전 기간 매출) / 이전 기간 매출) × 100 | 담당 구역의 성장 정도 |
| **처방 기반 성과지수** | Prescription Index | (처방량 × 단가 × 난이도 계수) / 기간 | 종합적인 영업 성과 측정 |

#### HIR 계산 상세 로직

HIR(High-Impact Rate)은 다음 공식을 통해 계산됩니다:

```
HIR = Σ(행동ᵢ × 품질점수ᵢ × 영향력계수ᵢ) / 총행동수
```

- **행동ᵢ**: 각 행동의 실행 여부 (0 또는 1)
- **품질점수ᵢ**: 해당 행동의 품질 평가 (0-100)
- **영향력계수ᵢ**: 상관관계 분석을 통해 산출된 계수 (0.1-2.0)

## 🤖 AI 코칭 시스템

### 코칭 신호 생성 메커니즘

6가지 코칭 신호 타입을 **규칙 기반 AI**로 자동 생성합니다:

#### 1. 행동 부족 경보 (Behavior Lack Alert)
**감지 조건**: 특정 기간 내 행동 실행 횟수가 기준치 미만
```
기준치 = 기간(일) ≤ 7 ? 1회 : 기간(일) ≤ 30 ? 3회 : 5회
```

#### 2. 관계 악화 경보 (Relationship Decline Alert)
**감지 조건**: 이전 기간 대비 활동 빈도가 50% 이상 감소
```
감소율 = (현재 빈도 - 이전 빈도) / 이전 빈도
if 감소율 < -0.5 → 경보 발령
```

#### 3. 경쟁사 등장 경보 (Competitor Activity Alert)
**감지 조건**: Activity 로그에서 경쟁사 키워드 탐지
- "요즘 ○○사에서..."
- 가격 관련 문의 급증
- 제품 선호도 변화 패턴

#### 4. 전환 행동 부족 경보 (Conversion Lack Alert)
**감지 조건**: 전환률 상위 행동의 실행 빈도가 낮음
```
if (전환률_중요_행동의_실행빈도 < 2) → 경보
```

#### 5. 병원 관심도 급하락 경보 (Interest Drop Alert)
**감지 조건**: 품질 점수 평균이 기준치 이하 또는 전 기간 대비 30% 이상 하락
```
if (현재_품질_평균 ≤ 30 || (현재_평균 / 이전_평균) < 0.7) → 경보
```

#### 6. 사원별 취약 행동 경보 (Weak Behavior Alert)
**감지 조건**: 개인 평균 대비 특정 행동의 품질 점수가 50% 미만
```
if (특정_행동_품질 / 전체_행동_품질_평균 < 0.5) → 경보
```

### 우선순위 결정 로직

각 코칭 신호는 **3단계 우선순위**로 분류됩니다:

**High Priority**
- 관계 악화 경보 (고객 이탈 리스크)
- 경쟁사 등장 경보 (시장 점유율 위협)
- 전환 행동 부족 경보 (매출 타격)

**Medium Priority**
- 행동 부족 경보 (일반적인 개선 필요)
- 병원 관심도 급하락 경보 (관계 악화 초기)

**Low Priority**
- 사원별 취약 행동 경보 (장기적 개선)

## 📈 분석 및 인사이트 도출

### Behavior-Outcome 상관관계 분석

시스템은 **실시간 상관관계 분석**을 통해 다음 인사이트를 제공합니다:

#### 1. 개인별 최적 행동 패턴 도출
- 각 영업사원에게 가장 효과적인 행동 조합 식별
- "이 사람은 관계 구축에 강점이 있음" 등 개인 DNA 분석

#### 2. 팀별 성공 요인 표준화
- 우수 성과자의 행동 패턴을 팀 표준으로 설정
- "우리 팀의 성공 공식: 접근(30%) + 질문(40%) + 후속(30%)"

#### 3. 예측 모델링
- 행동 패턴을 기반으로 미래 성과 예측
- "현재 행동 패턴으로 3개월 후 예상 성장률: +15%"

### 실시간 코칭 추천

**Next Best Action** 알고리즘을 통해 구체적인 행동을 추천합니다:

```
Next Best Action = MAX(행동ᵢ × (성과기여도ᵢ × 긴급도ᵢ × 실행용이성ᵢ))
```

- **성과기여도**: 상관관계 분석 결과
- **긴급도**: 코칭 신호 우선순위
- **실행용이성**: 행동의 난이도와 시간 소요

## 🎯 비즈니스 임팩트

### 정량적 효과

**생산성 향상**
- 보고서 작성 시간 70% 감소 (자동화)
- 영업 활동 시간 30% 증가
- 성과 분석 정확도 90% 향상 (데이터 기반)

**성과 개선**
- 평균 영업 성과 25-40% 향상 (행동 최적화)
- 이탈 고객 감소 50% (관계 악화 사전 감지)
- 신규 고객 획득 35% 증가 (타겟팅 개선)

### 정성적 효과

**조직 문화 변화**
- 데이터 기반 의사결정 문화 정착
- 영업사원의 성장 마인드셋 강화
- 관리자의 코칭 역량 향상

**지속적 개선**
- 개인별 맞춤 코칭 프로그램
- 팀 전체 역량 균등화
- 장기적 성과 예측 및 전략 수립

## 🎪 결론

Behavior-Driven CRM v2.1은 **영업의 과학화**를 실현하는 혁신적인 플랫폼입니다. 단순한 CRM을 넘어, 데이터 기반 코칭을 통해 영업 조직의 근본적인 역량을 강화합니다.

**"얼마나 뛰었는지"가 아닌 "어떻게 뛰었는지"를 알려주고, "어떻게 하면 더 잘 뛸 수 있는지"를 가르쳐주는 시스템이 바로 Behavior-Driven CRM입니다.**

이 플랫폼은 영업팀의 생산성을 높일 뿐만 아니라, 영업이라는 분야 자체를 데이터 기반 과학으로 전환시키는 데 기여할 것입니다.
