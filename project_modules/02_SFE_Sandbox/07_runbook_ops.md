# 07 Runbook Independent

## 목적
Sandbox를 독립 실행형 분석 엔진으로 운영할 때의 표준 절차를 정의한다.

## 일상 운영 절차
1. 입력 자산 확인
2. 시나리오 선택 또는 생성
3. 독립 실행
4. 품질 결과 확인
5. 시나리오 비교
6. 승인안 선택
7. OPS handoff 패키지 생성

## 재실행 기준
- 입력 자산 변경 후 재실행
- 시나리오 조건 변경 후 재실행
- 품질 게이트 실패 후 수정 재실행
- 승인 전후 결과 차이 검증 필요 시 재실행

## 장애 대응 플로우
1. 실패 stage 확인
2. 원인 분류
   - 입력 자산
   - 정규화
   - 조인
   - 분석
   - 승인 메타
   - handoff export
3. 조치 수행
4. 동일 시나리오 재실행
5. 변경 이력 기록

## 운영 체크포인트
- 실험 결과와 승인 결과 분리 저장
- 이전 승인 버전 덮어쓰기 금지
- handoff는 승인 완료 후에만 수행
